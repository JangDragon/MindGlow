##### AI를 활용하여 신속한 산재 가능성 판단으로 일반인들에게 도움을 주는 웹
###### 인원 : 5명
###### 기간 : 2024-05 ~ 2024-06

## 💼 주요 기능 소개

### 📑 자가 진단 테스트
간단한 테스트로 접근성을 높이고,
현재 자신의 상태가 심한지 판단을 할 수 있는 서비스

### ⁉️ 심리 센터 위치 안내
카테고리에 따라 자신에게 알맞는 센터 위치를 검색할 수 있으며,
선택한 센터의 간단한 정보 또한 알 수 있다.

### 📋 상담사 선택
해당 센터에 근무하는 상담사들의 정보를 보여주며,
선택한 상담사와 상담 연결을 도와준다.

### 📋 심리상담
오프로 상담 예약을 할 수 있지만,
채팅방을 활용하여 온라인에서도 상담할 수 있다.
